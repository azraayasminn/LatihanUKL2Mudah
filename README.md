# Program Menentukan Bilangan Ganjil atau Genap 

# Deskripsi
Program ini digunakan untuk menentukan apakah suatu **bilangan bulat** yang dimasukkan oleh pengguna adalah **ganjil** atau **genap**.  
Program juga melakukan **validasi input** agar hanya menerima bilangan bulat.

# Cara Kerja Program
1. Pengguna diminta memasukkan bilangan bulat.  
2. Program memeriksa apakah input valid (bukan huruf atau karakter lain).  
3. Jika valid, program memeriksa apakah bilangan tersebut **genap atau ganjil** menggunakan operator modulus `%`.  
4. Hasil ditampilkan ke layar.
